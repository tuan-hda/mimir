# Mimir Repo Brief

## Purpose
- `Mimir` is a personal engineering improvement portal.
- The product is not a generic article reader or news feed.
- The current direction is a calm planning surface where a user picks a topic, chooses a concrete task, confirms it, and manages a learning list.

## Current Product Shape
- Show focused engineering topics such as AI, frontend, Go backend, DevOps, and cloud.
- Show concrete things to do for the selected topic.
- Let the user select a task, confirm it into a learning list, and manage that list.
- Keep the interface minimal, quiet, and editorial rather than dashboard-heavy.

## Working Rules
- Commit directly on `main` for this repository when the user asks for a commit.
- Prefer small, reviewable changes that keep the app buildable after each commit.
- Preserve the current product direction: missions and actionable learning over feeds and filler content.
